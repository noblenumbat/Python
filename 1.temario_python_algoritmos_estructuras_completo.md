# Bloque 1. Sintaxis esencial de Python

## 1. Variables
Las variables permiten guardar información en memoria para usarla después.

### Ejemplo
```python
nombre = "Juan"
edad = 25
precio = 3500
```

### Explicación
- `nombre` guarda texto.
- `edad` guarda entero.
- `precio` guarda número.

### Ejercicios
1. Crear variables para nombre, edad y precio.
2. Cambiar el valor de una variable.
3. Mostrar variables con `print()`.

---

## 2. Entrada y salida
Permite recibir datos del usuario y mostrar resultados.

### Ejemplo
```python
nombre = input("Ingrese su nombre: ")
print(nombre)
```

### Explicación
- `input()` recibe texto.
- `print()` muestra en pantalla.

### Ejercicios
1. Pedir nombre al usuario.
2. Pedir edad y mostrar mensaje.
3. Pedir dos números.

---

## 3. Condicionales
Permiten tomar decisiones.

### Ejemplo
```python
edad = 20

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")
```

### Explicación
- `if` evalúa condición.
- `else` ejecuta alternativa.

### Ejercicios
1. Positivo o negativo.
2. Mayor de edad.
3. Comparar dos números.

---

## 4. Ciclos
Permiten repetir acciones.

### For
```python
for i in range(5):
    print(i)
```

### While
```python
contador = 0
while contador < 5:
    print(contador)
    contador += 1
```

### Explicación
- `for` recorre secuencia.
- `while` repite mientras condición sea verdadera.

### Ejercicios
1. Mostrar números del 1 al 10.
2. Tabla del 3.
3. Cuenta regresiva.

---

## 5. Funciones
Agrupan tareas reutilizables.

### Ejemplo
```python
def saludar():
    print("Hola")
```

### Uso
```python
saludar()
```

### Explicación
- `def` define función.

### Ejercicios
1. Función saludo.
2. Función suma.
3. Función descuento.

---

# Bloque 2. Algoritmos básicos más usados

## 1. Suma acumulada
Se usa para sumar varios valores.

### Ejemplo
```python
numeros = [1,2,3]
suma = 0

for n in numeros:
    suma += n
```

### Explicación
- `suma` acumula.

### Ejercicios
1. Sumar del 1 al 5.
2. Sumar lista.

---

## 2. Buscar máximo

### Ejemplo
```python
numeros = [3,8,2,10]
mayor = numeros[0]

for n in numeros:
    if n > mayor:
        mayor = n
```

### Explicación
Se compara cada elemento.

### Ejercicios
1. Mayor en lista.
2. Mayor deuda.

---

## 3. Contar ocurrencias

### Ejemplo
```python
contador = 0

for n in numeros:
    if n == 8:
        contador += 1
```

### Explicación
Cuenta coincidencias.

### Ejercicios
1. Contar número.
2. Contar vocales.

---

## 4. Búsqueda simple

### Ejemplo
```python
for n in numeros:
    if n == 2:
        print("Encontrado")
```

### Explicación
Busca elemento.

### Ejercicios
1. Buscar número.
2. Buscar producto.

---

## 5. Filtrado

### Ejemplo
```python
for n in numeros:
    if n > 5:
        print(n)
```

### Explicación
Selecciona datos según condición.

### Ejercicios
1. Números mayores a 5.
2. Deudas mayores a 3000.

# Bloque 3. Estructuras de datos en Python

## 1. Listas
Guardan múltiples valores.

### Ejemplo
```python
productos = ["arroz", "azúcar", "sal"]
```

### Operaciones
```python
productos.append("aceite")
productos.remove("sal")
```

### Ejercicios
1. Crear lista.
2. Agregar.
3. Eliminar.

---

## 2. Diccionarios
Guardan datos por clave y valor.

### Ejemplo
```python
cliente = {
    "nombre": "Ana",
    "deuda": 5000
}
```

### Consultar
```python
print(cliente["nombre"])
```

### Modificar
```python
cliente["deuda"] = 3000
```

### Ejercicios
1. Crear cliente.
2. Cambiar deuda.
3. Agregar teléfono.

---

## 3. Tuplas
Datos fijos.

### Ejemplo
```python
coordenada = (10,20)
```

### Explicación
No se modifica.

### Ejercicios
1. Crear coordenadas.
2. Mostrar posición.

---

## 4. Conjuntos
Eliminan repetidos.

### Ejemplo
```python
numeros = {1,2,3,3,4}
```

### Explicación
Resultado sin duplicados.

### Ejercicios
1. Crear conjunto.
2. Ver resultado.
